Projects<br><br>#### 📊 [Micro-Lending Data Platform](https://github.com/SakshamPix/microlending_data_platform)<br>A data-engineering and backend implementation of a peer-to-peer lending domain that models operational data in MySQL, transforms it into a reporting star schema, and serves cached analytics via FastAPI and Redis.<br>* **Key Engineering:** Multi-stage ETL pipelines, look-ahead pagination caching, relational modeling for users, wallets, and loan applications, and automated stored procedures for risk scoring.<br>* **Stack:** Python, FastAPI, MySQL, Redis, Docker, Pytest.<br><br>#### 🎬 Call the Shots: Movie Sim<br>An interactive mobile simulation game where users act as movie studio executives, managing budgets, greenlighting scripts, and controlling production variables to box-office success.<br>* **Stack:** Flutter, Dart.<br><br>#### 🗺️ [RouteSync AI](https://github.com/SakshamPix/routesyncai_app)<br>A localized routing and transit application that finds optimal walking paths between local campus buildings and transit stations without relying on external web mapping APIs.<br>* **Key Engineering:** Custom heap-based $A^*$ search algorithm optimization using a Haversine-distance heuristic across a localized pedestrian graph (716 local graph nodes, 785 walking edges).<br>* **Stack:** Python, Flask, Leaflet, OpenStreetMap Data.<br><br>#### 📝 [Scenius Storyboard](https://github.com/SakshamPix/Scenius_Storyboarding_Tool)<br>A modern, responsive scriptwriting and storyboarding web application built for video creators, writers, and directors to write interactive scripts and dynamically map metadata.<br>* **Key Engineering:** Relational syncing mapping text to structured database rows (1-to-many), stateless UUID-session token authentication via Spring interceptors, and automated schema generation.<br>* **Stack:** Java, Spring Boot 3.3, Hibernate, React (Vite), PostgreSQL, Docker.<br><br>---<br><br>### 🔍 Current Focus<br><br>* 🔭 **I’m
